# Xavier chrome extension

[WIP] Contol web pages in chrome from [Xavier](https://github.com/ewnd9/xavier) (remote control application)

## Install

```
$ npm install
```

## Development

```
$ node_modules/.bin/webpack --watch
```

## CLI for testing

```
$ node cli/cli
$ node cli/cli <command-id>
```
